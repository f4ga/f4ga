**What I built from scratch (MVP, fully working):**

| Component | Implementation |
|-----------|----------------|
| **LSM Engine** | MemTable (B‚ÄĎtree), SSTable with Bloom + range filter, Leveled compaction |
| **Value Log** | WiscKey + mmap ‚Üí zero‚ÄĎcopy reads for large values (>64 bytes) |
| **WAL + MANIFEST** | Write‚ÄĎahead log (CRC32) + manifest with fsync, crash recovery |
| **MVCC** | Snapshot isolation, inverted timestamps, tombstone support |
| **Transactions** | WriteBatch (atomic) + interactive optimistic transactions, explicit `ErrConflict` |
| **Write Stall Controller** | 3‚ÄĎlevel backpressure (Soft / Hard / MemTable stall), Prometheus metrics |
| **Column Families** | Isolated LSM instances + atomic cross‚ÄĎCF writes |
| **gRPC API** | Unary + server‚ÄĎstreaming Scan, reflection |
| **REST + WebSocket** | HTTP gateway, live key updates |
| **CLI** | Cobra‚ÄĎbased: get/set/del/scan/txn + interactive REPL |
| **Web UI** | React + TypeScript + Tailwind (embedded via `embed.FS`) |
| **Observability** | `/metrics` (Prometheus), `/health`, `/ready` |
| **DevOps** | Docker Compose, GitHub Actions (lint, test, license‚ÄĎcheck) |
